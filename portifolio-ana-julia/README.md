## 💻 Sobre o Projeto

Este é o meu portfólio pessoal e profissional, desenvolvido para centralizar meus principais projetos, experiências e habilidades técnicas de forma visual, moderna e interativa. O layout foi totalmente planejado para oferecer uma excelente experiência de usuário (UX), adaptando-se perfeitamente a qualquer dispositivo.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

* **React** (Biblioteca para construção da interface)
* **Vite** (Ferramenta de build ultra-rápida)
* **TypeScript** (Tipagem estática para maior segurança no código)
* **Tailwind CSS** (Estilização responsiva e utilitária)
* **Lucide React** (Pacote de ícones modernos)

---

## ✨ Funcionalidades

* **Design Responsivo & Grid Simétrico:** Cards de projetos perfeitamente alinhados e adaptáveis para celular, tablet e computador.
* **Internacionalização (Multi-idioma):** Suporte completo para troca de idioma instantânea entre **Português (PT)** e **Inglês (EN)**.
* **Modal de Detalhes:** Seção interativa que abre especificações técnicas, capturas de tela e links diretos para o código/demonstração de cada projeto.
* **Formulário de Contato Integrado:** Conectado diretamente ao Netlify Forms para recebimento de mensagens sem necessidade de backend complexo.
* **Acesso ao Currículo:** Botão dedicado para visualização rápida e download do currículo profissional.
